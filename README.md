# Productivity SaaS

A full-stack task manager built with React, Node.js and PostgreSQL.

## Live Demo
https://productivity-saas-silk.vercel.app

## Features

- User authentication
- Create tasks
- Complete tasks
- Delete tasks
- Progress tracking

## Tech Stack

Frontend:
- React
- TypeScript
- Vite

Backend:
- Node.js
- Express

Database:
- PostgreSQL
